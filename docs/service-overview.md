# Service Overview

## One-Line Summary
SundaeWatta helps users check whether nearby food trucks are likely open now.

## Core Value
- Fast map-based discovery
- Community live status reporting
- Structured request flow for new trucks and updates

## Main Flows
1. Start app from intro screen.
2. Explore map and open vendor detail.
3. Submit open/closed live report.
4. Submit registration or update request.

## Product Boundaries
- No settings tab in current release.
- No reviews feature in current release.
- No in-app moderation workflow.
