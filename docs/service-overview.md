# Service Overview

## One-Line Summary
`순대와따` is a mobile-first service for checking whether nearby sundae and gopchang trucks are open today.

## Core User Value
- See nearby trucks quickly on a map
- Check likely current status based on live reports
- Open truck details to view menu, price, phone, and reviews
- Submit live status reports
- Request new truck registration
- Request truck info updates

## Main Screens
- `랜딩`
  - Short intro copy
  - Clear CTA into the map experience
- `지도 보기`
  - Main discovery screen
  - Truck pins on the map
  - Legend for current status colors
- `트럭 추가`
  - New truck registration request
  - Request progress overview
- `설정`
  - Reserved for future user options

## Status Model
- `영업중`
- `영업종료`
- `확인 제보 필요`

`사장님 확인` is intentionally deferred for later redesign.

## Product Principles
- Mobile first
- Simple Korean copy
- Fast scanning over dense information
- Request-based data moderation
- Shared community value over full anonymity trust
